 ### 값 증가시키기
 ```
var gemCounter = 0

for i in 1 ... 15 {
    moveForward()
    if isOnGem {
        collectGem()
        gemCounter = gemCounter + 1
    }
    if isBlocked {
        turnRight()
    }
}
```
